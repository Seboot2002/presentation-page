Red social básica

Este es un proyecto en el que realizo una red social básica para experimentar con el framework Angular junto con Bootstrap para los estilos. Asimismo, se utilizó una base de datos hecha en MongoDB.


Esta pequeña red social tiene características como:

  • Publicación de imágenes con descripción.
  • Perfil de usuario con nombre de usuario e imagen de perfil junto con una lista de seguidores y seguidos; además, de tus propias publicaciones.
  • Feed en el cual se observan tus publicaciones y la de los seguidos en orden.
  • Sistema de mensajería básico.
